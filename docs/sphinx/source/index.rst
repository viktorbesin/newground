newground Documentation
============================================

This prototype (newground) is the implementation of a reduction from **non-ground** logic programs to **disjunctive** logic programs using Body-decoupled Grounding.
If you want to know the documetation of **Newground**, no need to leave, you are also at the right place!
The newground prototype is the next stage in the development for a unified Body-decoupled Grounding prototype.

The source code can be found in the GitHub repository newground_ (or the Fork newground-Fork_).
The experiments can be found in the GitHub-Experiments repository newground-Experiments_.
If you are new to this protype be sure to check-out the :ref:`installation-reference`.
